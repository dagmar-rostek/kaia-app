## SRH-Leitfaden Formatierung (Pflicht)

| Kriterium | Vorgabe |
|-----------|---------|
| Schriftart | Arial |
| Schriftgröße Fließtext | 11 pt |
| Schriftgröße Fußnoten | 10 pt einzeilig |
| Zeilenabstand | 1,5-zeilig |
| Ausrichtung | Blocksatz |
| Seitenränder | oben 2,5 cm · unten 2,0 cm · links 4,0 cm · rechts 2,0 cm |
| Seitennummern | Kopfzeile, zentriert, arabische Zahlen |
| Abbildungsunterschriften | 11 pt, linksbündig, Quelle 10 pt |
| Umfang Nettotext | 80 Seiten ± 10% (72–88 Seiten) |
| Mindestquellen | 50 wissenschaftliche Quellen |
| Zitierstil | DGPs/APA 4. Auflage |
| Einreichung | 3 Exemplare Klebebindung + digitale Version (USB oder CD) |
| Zitierweise KI-Tool | Nutzung von Claude Code in Eidesstattlicher Erklärung deklariert |

## Abweichungen vom Exposé

Das Exposé sah eine 5-teilige Kapitelstruktur vor. Die Thesis hat diese in 7 Kapitel ausgedehnt:
- **Kap. 4** wurde in drei thematische Detaildokumente aufgeteilt (4a Implementierung, 4b LLM-Eval, 4c Studiendesign)
- **Kap. 5 Ergebnisse** und **Kap. 6 Diskussion** wurden als eigenständige Kapitel ausgegliedert (im Exposé implizit)
- **Kap. 7 Fazit** als separates Abschlusskapitel hinzugefügt

Diese Erweiterungen sind im SRH-Leitfaden (§ 6.2.9: Zwischenbericht) zu dokumentieren und im Kapitel 1.3 (Aufbau der Arbeit) begründet.
